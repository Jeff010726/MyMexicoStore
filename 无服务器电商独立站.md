# 无服务器电商独立站

## Core Features

- Airwallex支付系统集成

- 订单管理后台

- 可视化页面编辑器

- 商品管理

- 客户管理

- 页面模板系统

- 前台页面动态渲染

- 真实API集成

- GitHub Pages自动部署
